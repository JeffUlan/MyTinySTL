MyTinySTL
=======
  这是一个小型的STL，为了练习数据结构、模板编程和c++11等特性，而做的一个练习。这个小型的STL实现了绝大部分STL中原有的函数、容器，也加入了一些c++标准库中的内容。以下是具体实现的内容。
